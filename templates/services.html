🧭 Layout Types in Movie Magic

🏠

Home Page

Serves as the landing page. New users are prompted to register or login. Logged-in users are redirected to the movie list.

🔐

Login / Register

Simple form layouts that allow users to securely create an account or login to an existing one.

🎥

Movies Page

Grid layout displaying movie posters, titles, and pricing, with a "Book Now" button for each.

🪑

Booking Page

Clean form interface where users can input seat numbers and see booked seats. Validates conflicts before booking.

📋

Dashboard

A user-specific layout showing all past bookings, including seat numbers, total cost, and time.

📖

About Page

Static informational layout describing the platform's mission, features, and purpose.

📦

Services Page

Explains all page layouts and services in a visual, user-friendly format to help users navigate with ease.
